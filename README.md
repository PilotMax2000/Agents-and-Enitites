# Agents-and-Enitites

Used packages;
1) AstralPathfindingProject - Pathfinding;
2) DOTween - movement animaiton;
3) Graphy - tool for showing FPS, RAM and other for quick performance checking;

Controls:
Alt+H - Show/Hide Graphy panel;
Q - spawn 1 unit;
W - spawn 10 units;
E - spawn 100 units;
R - delete all units on scene;
